# SlackBOT
This is a versatile SlackBot for messaging! A first of many Stardance Challenge projects!

### Prerequisites:

Ensure that you have the following installed on your system:

* **Node.js** (v22.22.3 or higher)
* **npm** (v10.9.8 or higher)
* **@slack/bolt** (v4.7.3 or higher)
* **dotenv** (v17.4.2 or higher)
* **axios** (v1.17.0 or higher)

*** Installation

```bash
# Clone the repository
git clone https://github.com/JairusTumacder/SlackBOT.git

# Navigate into the project directory
cd slackBOT

# Install required npm packages
npm install @slack/bolt dotenv
```

---

### Usage

To use SlackBOT, run the index.js file in the terminal of your IDE

```bash
node index.js
```
